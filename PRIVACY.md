# Privacy

TokenGlance is local-first. It stores normalized token usage metadata under:

```text
~/Library/Application Support/TokenGlance/
```

TokenGlance does not upload usage data, prompts, responses, source code, credentials, browser cookies, shell history, clipboard contents, or network traffic.

Collectors parse only token metadata fields required for aggregation. Raw content in source records is not mapped into domain models, persisted, exported, or copied into diagnostics.
